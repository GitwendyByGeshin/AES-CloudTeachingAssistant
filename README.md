# AES智能云上授课助手 1.0 预览版

> 由智谱清言与ScriptCat及创作者绫白共同驱动

## 软件介绍

针对**出头科技**的**云上教学系统量身定制**的**全自动刷课解决方案**。突破系统限制，解放你的双手

## 核心优势
相较于市面上传统且容易失效的油猴脚本，AES智能云上授课助手在**底层防御突破、稳定性、安全性、交互体验**上进行了全面重构，具备以下无可比拟的优势：

1.**纯净无侵入设计（安全可靠）**
- **纯 API 驱动**：全程基于标准浏览器 API 与安全的**原型链扩展实现**，**不暴力修改或覆盖平台原有的业务逻辑代码**
- **零污染运行**：UI 面板采用独立容器挂载与严格的作用域 CSS 隔离，**不污染原有页面布局**，**不会导致系统页面错位或样式丢失**
- **功能无损兼容**：**完全不影响系统的其他正常功能**（如做笔记、随堂测验、章节切换等），脚本只专注于视频播放控制，真正做到“**用完即走，不破坏原有生态**”

2.**硬核反暂停机制（真实后台挂机）**
- **欺骗浏览器层级**：不仅监听页面事件，更在底层直接重写 document.hidden 和 document.visibilityState 属性，让系统误以为用户始终停留在当前页面
- **全维度事件拦截**：精准拦截 visibilitychange、blur、mouseleave 等可能导致视频暂停的事件，真正做到“**切走不暂停，息屏照常播**”

3.**铁壁级倍速防护锁（拒绝被系统限速）**:
- **原型链级劫持**：传统脚本只能被动对抗倍速重置，本脚本直接在 HTMLVideoElement.prototype 层面对 playbackRate 的 getter/setter 进行重写
- **智能降级保护**：在原型链劫持失败（极少数特殊情况）时，自动无缝切换至 ratechange 事件监听降级方案。双管齐下，死死锁住你设置的播放倍速（支持 1x - 3x 自由切换）

4.**智能路由与全自动连播**
- **SPA 架构完美适配**：精准监听 Hash 路由变化，完美适配出头科技的前端单页应用（SPA）架构，**无需刷新页面**即可无缝切换视频
- **双保险切课策略**：自动点击下一课时，若触发拦截失效，将在 500ms 内自动执行强制 Hash 跳转，确保 100% 成功切入下一课

5.**极致美学与沉浸式交互面板**
- **无视 CSP 拦截**：采用纯 JS 字符串注入配合 GM_addStyle 的安全模式，完美绕过严苛的 CSP（内容安全策略）导致的面板样式丢失问题
现代毛玻璃 UI：采用渐变暗黑主题，圆角卡片设计，支持**一键折叠**(折叠后化为小圆点，丝毫不遮挡视线）
- **数据全景监控**：面板内实时显示当前播放进度条、精确到秒的时间轴、当前课时名称、全局“已完成/剩余/总进度”统计，进度一目了然

6.**强大的环境兼容与容错能力**
- **动态播放器捕获**：使用 MutationObserver 动态监听 DOM 树，无论视频元素是懒加载还是延迟渲染，都能精准捕获；甚至支持**穿透跨域限制**尝试抓取 iframe 内的视频
- **断流自动重连**：内置视频 Error 事件监听，遇到网络波动导致播放失败时，**自动执行 load() 重载并恢复播放**，**无需人工干预**
- **静音破局**：针对现代浏览器严格的“自动播放策略（要求用户必须交互）”，脚本默认开启**强制静音**，以此绕过浏览器限制，实现打开页面即刻**自动播放**
## 安装方法

### 1. 安装 ScriptCat/油猴脚本 扩展
- Chrome: 从 Chrome 扩展组件搜索 "ScriptCat"/"油猴脚本" 安装
- Edge: 从 Edge 加载项商店搜索 "ScriptCat"/"油猴脚本" 安装
- Firefox: 从 Firefox 附加组件商店搜索 "ScriptCat"/"油猴脚本" 安装

### 2. 安装脚本
1. 打开 ScriptCat 扩展面板
2. 点击 "新建脚本"
3. 将 `AES-云上刷课助手.user.js` 文件内容粘贴进去
4. 保存并启用脚本

## 使用方法

### 基本使用
1. 登录云上教学系统: https://XXX.XX.superchutou.com/
2. 进入任意课程的视频播放页面
3. 脚本会自动开始播放视频
4. 右上角显示控制面板

### 控制面板功能

| 按钮 | 功能 |
|------|------|
| ▶/⏸ | 播放/暂停视频 |
| ⏮ | 上一课 |
| ⏭ | 下一课 |
| 🔊/🔇 | 静音切换 |
| 1x~3x | 播放速度选择 |

### 面板信息显示

- **运行状态**: 显示当前是否在播放
- **进度条**: 当前视频播放进度
- **视频标题**: 当前播放的视频名称
- **统计信息**: 已完成/剩余/总进度

## 注意事项

⚠️ **播放速度建议**
- 建议使用 1.5x 或 2x 速度
- 过高速度可能被检测异常

⚠️ **后台播放**
- 脚本已绕过页面焦点检测
