# AES智能云上刷课助手 1.0 预览版

> 由智谱清言与ScriptCat及创作者绫白共同驱动

## 软件介绍

AES智能云上刷课助手是专为出头科技云上教学系统设计的自动刷课工具，具有可视化控制面板，支持后台自动播放、倍速播放、自动切换课程等功能。

## 功能特性

✅ **自动播放** - 进入视频页面自动开始播放  
✅ **倍速播放** - 支持1x/1.25x/1.5x/2x/3x播放速度  
✅ **后台播放** - 切换标签页后继续播放  
✅ **自动静音** - 避免后台播放时声音干扰  
✅ **自动下一课** - 视频播放完成后自动切换下一个视频  
✅ **可视化面板** - 精美控制面板，实时显示进度  
✅ **拖拽移动** - 面板支持拖拽移动位置  
✅ **状态通知** - 播放进度和状态变化通知  

## 安装方法

### 1. 安装 ScriptCat 扩展
- Chrome: 从 Chrome 网上应用店搜索 "ScriptCat" 安装
- Edge: 从 Edge 加载项商店搜索 "ScriptCat" 安装
- Firefox: 从 Firefox 附加组件商店搜索 "ScriptCat" 安装

### 2. 安装脚本
1. 打开 ScriptCat 扩展面板
2. 点击 "新建脚本"
3. 将 `AES-云上刷课助手.user.js` 文件内容粘贴进去
4. 保存并启用脚本

## 使用方法

### 基本使用
1. 登录云上教学系统: https://csjs.web2.superchutou.com/
2. 进入任意课程的视频播放页面
3. 脚本会自动开始播放视频
4. 右上角显示控制面板

### 控制面板功能

| 按钮 | 功能 |
|------|------|
| ▶/⏸ | 播放/暂停视频 |
| ⏮ | 上一课 |
| ⏭ | 下一课 |
| 🔊/🔇 | 静音切换 |
| 1x~3x | 播放速度选择 |

### 面板信息显示

- **运行状态**: 显示当前是否在播放
- **进度条**: 当前视频播放进度
- **视频标题**: 当前播放的视频名称
- **统计信息**: 已完成/剩余/总进度

## 注意事项

⚠️ **播放速度建议**
- 建议使用 1.5x 或 2x 速度
- 过高速度可能被检测异常

⚠️ **后台播放**
- 脚本已绕过页面焦点检测
- 建议偶尔切回页面确认播放正常
